# Dateinamen
- Snake-Case für Dateinamen verwenden. 
- Kleinschreiben
- .rs als Suffix
- Z.b. hello_world.rs

- Alle Codedateien müssen ind em Ordner *src* liegen
- ReadMe, Lizenzbestimmungen ua. gehören auf die Toplevel Ebene, ebenso Cargo.toml

# Codestyle
- Hinter Funktionsnamen eine Leerzeichen dann {}
- Die öffnende { auf der sleben Zeile wie der Funktionsname
- Lange Methoden-Ketten umbrechen und Einrücken
- Für Konstanten alle Buchstaben in Großbuchstaben und die Wörter mit dem Unterstrich trennen
- Snake Case für Methoden Namen
- Snake Case für Variablen Namen
- Kommentare auf der selben Zeile wie der Code benötigen nur ein Leerzeichen als Trenner z.B. *let x = 1 // x ist Eins*
- Sätze in Kommentaren starten mit Großbuchstaben, weitere Zeilen bis Satzende mit Kleinbuchstaben.
